# POO-Conceitos-Paradigmas-Programacao

Trabalho universitário da disciplina de Conceitos de Linguagens de Programação. Este projeto demonstra os conceitos fundamentais da Programação Orientada a Objetos (POO) utilizando a linguagem C#.

## Descrição

Este repositório contém exemplos práticos e implementações dos principais paradigmas da POO, incluindo:
- Classes e Objetos
- Herança
- Polimorfismo
- Encapsulamento
- Abstração

## Estrutura do Projeto

- `ProjetoPOO/`: Pasta principal do projeto C#
  - `Program.cs`: Arquivo principal com exemplos de código
  - `ProjetoPOO.csproj`: Arquivo de configuração do projeto

## Como Executar

1. Certifique-se de ter o .NET 6.0 ou superior instalado.
2. Navegue até a pasta `ProjetoPOO`.
3. Execute o comando: `dotnet run`
